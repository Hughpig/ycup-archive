# About

## 介绍

YCup 是一个旨在服务上海市曹杨第二中学学生的公益性组织，致力于提供高质量的数学训练资源并举办线下年级比赛。

## 条款

以下条款适用于参加 YCup 的所有参赛者。

参赛者应自觉遵守校纪校规、公序良俗和法律法规。组委会保留对违反该条例的选手永久的、不可撤销的夺权权力。

参赛者可以提交数学答案，并保留提交答案中所有知识产权和工业产权（包括道德权利）的所有权。作为提交的条件，参赛者授予 YCup 委员会一项永久的、不可撤销的、全球性、免版税且非独占的许可，允许其使用、复制、改编、修改、出版、分发、公开展示、创作衍生作品及公开展示。

参赛者以题目自身为基础提交答案，不作任何明示或暗示的保证或条件，包括但不限于任何标题、非侵权、可销性或特定用途适用性的保证或条件。
